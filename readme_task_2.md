# Sentiment Analysis using TF-IDF and Logistic Regression

## Internship Task 2 – Machine Learning

This project performs **Sentiment Analysis on customer reviews** using **Natural Language Processing (NLP)** techniques. The model classifies reviews as **positive or negative** using **TF-IDF vectorization and Logistic Regression**.

---

## Objective

Perform sentiment analysis on a dataset of customer reviews using:

- TF-IDF Vectorization
- Logistic Regression

---

## Concepts Covered

- Natural Language Processing (NLP)
- Text Preprocessing
- Stopword Removal
- TF-IDF Vectorization
- Logistic Regression
- Sentiment Classification

---

## Dataset

The dataset contains **customer reviews** and their corresponding **sentiment labels**.

Columns in the dataset:

- review → Customer review text  
- sentiment → Sentiment label (positive or negative)

Example:

| Review | Sentiment |
|------|------|
| Excellent product and very useful | Positive |
| Waste of money | Negative |

---

## Technologies Used

- Python
- Jupyter Notebook
- Pandas
- Scikit-learn
- NLTK
- NumPy

---

## Project Workflow

1. Import required libraries  
2. Load the dataset (reviews.csv)  
3. Perform text preprocessing  
4. Convert text into numerical features using TF-IDF  
5. Split dataset into training and testing sets  
6. Train Logistic Regression model  
7. Predict sentiments  
8. Evaluate model performance  

---

## Model Evaluation

The model performance is evaluated using:

- Accuracy Score
- Confusion Matrix
- Classification Report
