# Decision Tree Implementation using Scikit-learn

## Internship Task 1 – Machine Learning

This project demonstrates the implementation of a **Decision Tree Classifier** using the **Scikit-learn library in Python**. The objective of this task is to build and visualize a machine learning model that can classify outcomes based on a dataset.

---

## Objective

Build and visualize a **Decision Tree model** to classify or predict outcomes on a chosen dataset.

---

## Concepts Covered

- Machine Learning Basics
- Supervised Learning
- Decision Tree Algorithm
- Data Splitting (Training and Testing)
- Model Evaluation
- Decision Tree Visualization

---

## Dataset

This project uses the **Iris Dataset**, which contains measurements of iris flowers including:

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

The goal is to classify the flower species into:

- Setosa
- Versicolor
- Virginica

---

## Technologies Used

- Python
- Jupyter Notebook
- Scikit-learn
- Pandas
- NumPy
- Matplotlib

---

## Project Workflow

1. Import required libraries  
2. Load the Iris dataset  
3. Explore and understand the dataset  
4. Split the dataset into training and testing sets  
5. Train the Decision Tree model  
6. Make predictions on test data  
7. Evaluate model performance  
8. Visualize the decision tree  

---

## Model Evaluation

The model performance is evaluated using:

- Accuracy Score
- Confusion Matrix
- Classification Report

---

## Conclusion

This project demonstrates how **Decision Tree algorithms** can be used for classification tasks. Decision trees are easy to understand and widely used in machine learning applications.
