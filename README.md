# Machine Learning Internship Projects

This repository contains the implementation of **four Machine Learning tasks** completed during the internship.  
Each task focuses on different machine learning and data science concepts including classification, NLP, deep learning, and recommendation systems.

---

# Task 1: Decision Tree Implementation

## Objective
Build and visualize a **Decision Tree model** using Scikit-learn to classify or predict outcomes on a chosen dataset.

## Concepts Covered
- Machine Learning Basics
- Supervised Learning
- Decision Tree Algorithm
- Model Training and Evaluation
- Data Visualization

## Technologies Used
- Python
- Scikit-learn
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Workflow
1. Import required libraries
2. Load the dataset (Iris dataset)
3. Perform data exploration
4. Split dataset into training and testing sets
5. Train the Decision Tree classifier
6. Evaluate the model using accuracy and confusion matrix
7. Visualize the decision tree structure

## Output
- Decision Tree visualization
- Model evaluation metrics

# Task 2: Sentiment Analysis using NLP

## Objective

## Concepts Covered
- Natural Language Processing (NLP)
- Text Preprocessing
- Stopword Removal
- TF-IDF Vectorization
- Logistic Regression
- Sentiment Classification

## Technologies Used
- Python
- Pandas
- Scikit-learn
- NLTK
- NumPy
- Jupyter Notebook

## Workflow
1. Import libraries
2. Load dataset containing customer reviews
3. Perform text preprocessing
4. Convert text into numerical features using TF-IDF
5. Train Logistic Regression model
6. Predict sentiments
7. Evaluate model performance

## Output
- Sentiment classification (Positive / Negative)
- Model evaluation metrics
